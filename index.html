Desenvolvedor Front-End


  O Desenvolvedor Front-End é o profissional responsável pela interface de um projeto web, ou seja, é ele quem projeta, estrutura e facilita o desenvolvimento. A capacidade de produzir algo que seja criativo e, ao mesmo tempo, forneça uma boa experiência ao Desenvolvedor Front-End você aprenderá desde o básico até o avançado da linguagem JavaScript e ES6 Essencial, além também de adquirir conhecimentos em ReactJS, Arquitetura de Sistemas e Git. Após concluir os 13 cursos do bootcamp, você receberá o certificado que te tornará especialista em Desenvolvedor Front-End.


  Programador Front-End tem por objetivo o desenvolvimento de competências que possibilitem ao concluinte programar sistemas utilizando as principais tecnologias do front-end, disponibilizando visualmente ao usuário uma forma amigável de acesso ao sistema, utilizando tecnologias como HTML, CSS e JavaScript e considerando as normas, padrões e requisitos técnicos, de qualidade, saúde e segurança e de sustentabilidade.


Objetivo


  Desenvolver habilidades básicas sobre programação front-end, utilizando as linguagens de marcação HTML5 e CSS e as boas práticas de desenvolvimento web, para a construção, organização e personalização de páginas web simples.

O que é HTML?


  Sigla para HyperText Markup Language — Linguagem de Marcação de Hipertexto —, o HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos. Os hipertextos são a junção de vários elementos — palavras, vídeos e conteúdos —, que, quando conectados, estabelecem uma rede de dados que permite a comunicação, o armazenamento e o compartilhamento de informações.

O que é JAVASCRIPT


  O JavaScript é uma linguagem de programação que possibilita a implementação de objetos complexos em páginas web. Conteúdos em páginas que se atualizam constantemente (sites de notícias, por exemplo), informações interativas em gráficos, animações 2D ou 3D, etc. Tudo isso só é possível graças ao JavaScript.


O que é React


  React é um framework JavaScript criado pelo Facebook (atual Meta) que é usado para criar interfaces de usuário (UI) em aplicativos web. Ele é popular por ser fácil de usar, altamente flexível e escalável, e é usado por muitas empresas de tecnologia
